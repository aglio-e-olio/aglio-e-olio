## **Aglio Olio는 개발자 취준생에게 크게 두 가지 경험을 제공합니다**
저희 프로젝트는 다른 누구의 문제도 아닌 ‘개발자 취준생인 나’의 문제를 해결하는 데 집중했습니다.

- 타인과의 원활한 실시간 스터디 경험
- 내가 저장한 스터디를 언제든지 회고할 수 있는 경험

이 두 가지 경험을 통해 온라인 코딩 스터디의 시작과 끝을 함께합니다.
